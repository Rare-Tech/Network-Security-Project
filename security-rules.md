# Security Rules

## Firewall Rules
- Allow IT VLAN full access to all networks
- Block HR VLAN from accessing Admin VLAN
- Block Admin VLAN from accessing HR VLAN
- Allow Guest VLAN internet access only
- Deny all unknown inbound traffic

## Security Measures
- Network segmentation using VLANs
- Access control between departments
- Firewall filtering traffic between subnets
